## 1 - Create API Key from [Google AI Studio](https://aistudio.google.com/)
## 2 - Add the key as an environment variable `GOOGLE_API_KEY=<YOUR_API_KEY>`
## 3 - Install dependencies specified in `requirements.txt`:
```
pip install -r requirements.txt
```
## 4 - run the following script
```
python audio-to-audio-liveapi.py
```
